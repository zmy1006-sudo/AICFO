## ⚠️ 已知限制

| 限制项 | 说明 | 应对方案 |
|-------|------|---------|
| 体验模式凭证上限3条 | 免注册用户只能查看3条演示凭证 | 引导用户注册正式账号 |
| 发票识别仅支持增值税普通/专用发票 | 暂不支持定额发票、卷式发票等 | 后续版本迭代 |
| 银行流水仅支持Excel/CSV | 不支持PDF格式银行回单 | 建议导出Excel格式 |
| 报税结果仅供参考 | 一键算税结果不等于最终申报数据 | 提示用户以电子税务局为准 |
| 地区覆盖有限 | MVP仅覆盖北京地区税务规则 | Sprint 2扩展至全国主要城市 |
| AI置信度显示 | 复杂业务AI置信度可能偏低 | 低于80%时提示用户人工核对 |
| 数据导出功能缺失 | V0.1暂不支持导出 | Sprint 2优先级功能 |
